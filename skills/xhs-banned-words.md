# 小红书医美内容创作违禁词与合规替换词对照表

在小红书进行医疗美容相关内容创作时，平台审核极其严格。本文档整理了最全面的违禁词库及其合规替换表达，涵盖绝对化用语、医疗功效宣称、安全承诺等多个高风险领域。创作者必须严格遵守这些规范，避免笔记被限流、下架或账号被封禁。

## 绝对化与极限用语

绝对化用语是广告法和小红书平台明确禁止的表述，这类词汇通常带有强烈的夸大和不可验证的承诺性质。在任何情况下，都应避免使用此类词汇。

| 违禁词类别 | 具体违禁词汇 | 合规替换表达 |
| --- | --- | --- |
| 极致程度词 | 最、最佳、最好、最强、最优、顶级、极致、绝对、无敌、No.1 | 很好用、口碑很好、强烈推荐、体验不错 |
| 唯一性宣称 | 第一、唯一、首个、独家、独一无二、全网最低、史上最强、业界第一 | 值得一试、特色项目、人气很高 |
| 绝对效果承诺 | 100%、百分百、一定、肯定、保证、必定、绝对有效、零风险、无副作用 | 亲测有效果、很多人反馈不错、安全性高 |
| 虚假时间承诺 | 立竿见影、立即见效、秒变、一次见效、一用就白、三天见效 | 坚持使用有变化、按疗程进行、逐渐改善 |
| 永久性承诺 | 永久、终身、终身维持、永不褪色、永不变形 | 维持时间因人而异、长效保持、效果持久 |

## 医疗与疾病治疗宣称

医美项目虽然具有医疗属性，但在小红书等社交平台作为非专业医疗内容发布时，必须弱化"治疗疾病"的色彩，转向"改善状态"的美容分享。

| 违禁词类别 | 具体违禁词汇 | 合规替换表达 |
| --- | --- | --- |
| 疾病治疗 | 治疗、疗效、医治、治愈、根治、康复、痊愈、恢复健康 | 改善、缓解、修护、帮助调整 |
| 抗菌消炎 | 消炎、杀菌、抗病毒、抗菌、抑菌、除菌、灭菌、防菌 | 舒缓泛红、镇定肌肤、清洁修护 |
| 医疗术语 | 处方、药方、药用、药物、中药、临床验证、医学证明 | 专业配方、科学成分、经过测试 |
| 权威背书暗示 | 医生推荐、专家认证、权威机构、FDA认证、药监局批准 | 专业人士建议、正规渠道引进、合规产品 |
| 特定病症名称 | 皮肤病名称、各类具体疾病名称、抗癌、防癌 | 肌肤问题、亚健康状态、日常保养 |

## 身体形态与面部改善

涉及身体形态改变或面部特定问题改善的词汇，往往容易触碰"虚假宣传"或"制造焦虑"的红线。应使用更温和、主观的体验式表达。

| 违禁词类别 | 具体违禁词汇 | 合规替换表达 |
| --- | --- | --- |
| 减脂塑形 | 减肥、瘦身、燃脂、溶脂、吸脂、瘦脸、瘦腿 | 身材管理、线条紧致、体态优化、视觉显瘦 |
| 丰胸增高 | 丰胸、增高、壮阳、补肾 | 曲线塑造、体态改善、日常保养 |
| 肤色与斑点 | 排毒、祛斑、美白、斑立净、无斑 | 提亮肤色、改善暗沉、均匀肤色、肌肤透亮 |
| 痘痘与疤痕 | 祛痘、祛疤、除疤、平坑 | 改善痘痘肌、淡化痘印、修护受损肌肤 |
| 毛发相关 | 生发、毛发再生、止脱、根治脱发 | 强韧发丝、滋养头皮、改善掉发困扰 |

## 成分与技术夸大

对于医美产品或项目的成分及技术原理，不能进行无科学依据的夸大，也不能使用违规的化学或生物学名词。

| 违禁词类别 | 具体违禁词汇 | 合规替换表达 |
| --- | --- | --- |
| 违规成分名称 | 激素、抗生素、药物成分、美白针、溶脂针、胎盘素、干细胞 | 营养成分、透明质酸、胶原蛋白、合规针剂 |
| 虚假成分宣称 | 100%纯天然、无任何化学成分、纯天然产品 | 植萃成分、成分温和、自然提取 |
| 伪科技名词 | 纳米、基因、太空、黑科技、神器、奇迹、神效 | 创新技术、专利配方、宝藏好物 |
| 夸张前缀修饰 | 特效、全效、强效、奇效、速效、超强、全面、全方位 | 多效合一、针对性改善、综合护理 |
| 传统秘方暗示 | 祖传、御制、秘制、宫廷、百年老字号、偏方、秘方 | 经典配方、传统工艺、特色配比 |

## 平台"三品一械"特别禁推令（2026版）

根据小红书最新的内容管理规范，对于医疗美容相关的特定产品和项目，有更严格的展示和描述限制。

| 禁推类别 | 具体限制内容 | 合规创作建议 |
| --- | --- | --- |
| 处方药与耗材 | 全面禁止推广处方药；禁止展示注射针头、注射器等医疗耗材。 | 侧重分享面诊体验、医生沟通细节，避免直接展示医疗器械。 |
| 三类医疗器械 | 如各类光电设备（热玛吉、超声炮等），禁止以"产品名义"进行直接推广和售卖。 | 以"抗衰老思路"、"紧致提升经验"为切入点，分享个人护理心得。 |
| 案例对比图 | 禁止发布具有明显诱导性的术前术后直接对比图。 | 使用不同角度、不同光线的氛围感照片，或分享恢复期的心路历程。 |
| 具体项目名称 | 在纯种草内容中，尽量避免直接提及具体的医美项目、仪器或药品名称。 | 聚焦于"解决什么问题"（如：如何改善法令纹），而非"卖什么项目"。 |
| 机构与医生 | 禁止使用"最好医院"、"顶级专家"、"韩国医生"（无资质证明）等夸大表述。 | 强调"正规机构"、"执业医师"、"面诊沟通的重要性"。 |
