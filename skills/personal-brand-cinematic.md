# 个人品牌电影感视觉打造指南

专为企业家、创始人、行业专家设计的视觉形象系统。通过电影摄影技术，将个人品牌定位转化为可落地的拍摄方案。

## 核心理念

**视觉即品牌语言** — 不同的电影风格传递不同的个人特质：

| 风格 | 传递特质 |
|---|---|
| 史诗感 | 格局与远见 |
| 文艺感 | 温度与故事 |
| 纪录片 | 真实与专业 |
| ... | ... |

---

## 使用流程

1. **明确个人品牌定位**：行业权威 / 亲和温度 / 创新前沿 / 沉稳大气
2. **匹配拍摄场景**：根据定位选择核心场景（见下方场景库）
3. **确定电影风格**：为每个场景选择最佳的电影摄影风格
4. **生成AI提示词**：将配置转换为可用于 Midjourney/Stable Diffusion 的视觉提示词

## 创始人形象场景库

### 权威专业场景

**场景1：演讲/发布会**
品牌传递：行业领袖、思想前瞻性

| 方案 | 配置 | 特点 |
|---|---|---|
| 首选：史诗感宽银幕 | Red V-Raptor + Panavision C-series 变形镜头，35-50mm | 椭圆光斑、蓝色炫光、极阔视野 |
| 备选：现代商业大片 | Sony Venice + Zeiss Ultra Prime，24-35mm广角 | 高动态范围、通透感、大制作质感 |

拍摄要点：低角度仰拍增强权威感，背景虚化处理，侧面光勾勒轮廓

**场景8：电影海报风格形象广告（专业级）**
品牌传递：定义性视觉形象、艺术院线质感、精英气质

适用场景：个人品牌核心视觉、年度形象大片、高端宣传物料

**造型设计（先锋高级时装）：**
- 主体：浅卡其/米灰色三件套西装（宽松廓形外套 + 紧身马甲 + 高腰阔腿裤）
- 配饰：深色大框太阳眼镜 + 方形印章戒指
- 哲学：正式与松弛的矛盾并置，传递"昂贵、原创、触感深刻"

**灯光设计：**
- 主光：45度侧上方硬光，制造明暗对比（Chiaroscuro）
- 辅助光：轮廓光从背后勾勒人物边缘
- 整体氛围：低照度 + 高对比，营造沉稳、精英、略疏离的情绪

**色彩调色板：**
- 主色：深灰（背景）+ 浅卡其/米灰（西装）
- 后期：模拟胶片色彩科学（Kodak Portra或Cinestill风格），避免过度饱和

## 多平台内容适配

**小红书/抖音/视频号（9:16竖版）：**
- 演讲/访谈类：从中景改为近景（头部+肩膀），保留电影感虚化效果
- 工作/手术类：特写镜头为主（手部、工具、产品）
- 环境/机构展示：上下分屏或慢速垂直摇镜头

**公众号/官网/封面（16:9横版）：**
- 人物位于画面黄金分割点（左/右1/3处）
- 背景保留30-40%空间放文字
- 眼神方向留出"想象空间"

## 进阶：建立视觉识别系统

1. **固定配色方案**：主色调与品牌色一致，辅助色通过灯光和后期统一
2. **标志性构图**：特定角度成为你的"视觉签名"（如总是从左侧45度拍摄）
3. **一致的光影风格**：侧逆光勾勒轮廓，或柔和漫光展现亲和
4. **标志性元素**：特定道具（医美产品、行业奖项、专利证书）

**系列化内容规划：**
- 技术篇：手术过程（现代商业风格）
- 观点篇：行业访谈（怀旧文艺风格）
- 日常篇：工作花絮（纪录片风格）
- 愿景篇：演讲展示（史诗感风格）
